# Backend Tutorial

|  |  |
| --- | --- |
|  | This tutorial uses A12 version 2025.06-ext5 and is based on the Project Template version 202506.5.1. |

## Prerequisites

For this introductory tutorial, prior knowledge of the following technologies is necessary:

* Java
* Spring Boot

You need to have the following programs available:

* Text editor / IDE
* [A12 Modeling Environment](https://geta12.com/installer/) in version **2025.06-ext5**

We will extend the CRM system, which has been created and implemented in the previous tutorials.
If you want to know how it was implemented, you can check them out.

## Overview

This tutorial serves as an introduction into customizing the A12 backend.

The following aspects will be covered in the tasks:

* [Task 1: Document Access](https://geta12.com/docs/overall/dev_tutorial_backend_document_access/index.html)
* [Task 2: Document Manipulation](https://geta12.com/docs/overall/dev_tutorial_backend_document_manipulation/index.html)
* [Task 3: Unit Testing](https://geta12.com/docs/overall/dev_tutorial_backend_unit_testing/index.html)
* [Task 4: Custom Endpoint](https://geta12.com/docs/overall/dev_tutorial_backend_custom_endpoint/index.html)
