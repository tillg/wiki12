# Get Started with Modeling

Welcome to the A12 Modeling Environment. By the end of this guide, you’ll know how to access the tools included in your installation and explore the delivered Sample Workspaces in both, the Simple Model Editor and the Preview App.

## Download and Install

To download and install the A12 Modeling Environment, please follow the instructions from [Installing A12](https://geta12.com/docs/overall/installing_a12/index.html).

### What Is Included in the A12 Installation

When you install the A12 Modeling Environment, you’ll gain access to the following tools and components:

* The *[Simple Model Editor](https://geta12.com/docs/sme/sme-ba-docs/index.html)*: The SME is the primary tool for creating and editing models.
* The *Preview App Control*: A utility to manage the *[Preview App](https://geta12.com/docs/overall/preview_app/index.html)* and workspaces. The Preview App allows you to test your models in a locally running full stack application.
* *Sample Workspaces*: Pre-configured example workspaces that demonstrate A12’s modeling capabilities and provide a starting point for your own projects.

Additionally, you can find the *Camunda Modeler* to work with workflows models (bpmn & dmn) and a link to the documentation.

## Exploring the Modeling Environment

After installing, locate the tools on your system. You can either open the respective directory from the installer application or navigate to the defined directory on your filesystem. If the name was not changed during installation its default is "A12 Tools - release line version". Within it you will find the following:

![directory](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/directory.png)

Figure 1. Directory Content

Start the Preview App Control and choose one of the offered Sample Workspaces by clicking into the "Workspace"-Select:

![select sample workspace](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/select_sample_workspace.png)

Figure 2. Select a Sample Workspace

The installation includes Sample Workspaces, which are pre-configured examples to help you understand modeling features. After selection, simply click the "Start"-Button which will trigger the start of a Preview App Client and Server for you locally.

This may take some time, depending on the selected workspace, but when finished your default browser will open and show you the login screen of the Preview App.

![previewapp login](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/previewapp_login.png)

Figure 3. Preview App Login

You can log in with "enduser" and password "a12". This is true for any of the delivered Sample Workspaces.

![pa basic](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/pa_basic.png)

Figure 4. Preview App with the Basic Workspace

![pa e comm](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/pa_e-comm.png)

Figure 5. Preview App with the e-Commerce Workspace

Within the Preview App, you can now navigate via the main menu to explore the different modules and its functionality.

![navigate inPA](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/navigate_inPA.png)

Figure 6. Navigate through the Preview App…​

![navigate inPA2](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/navigate_inPA2.png)

Figure 7. …​and see different layouts and features in the different Sample Workspaces.

You can stop a running Preview App using the Preview App Control and switch between the Sample Workspaces.

To view the corresponding models for the running application, start the SME and select the same workspace you chose in the Preview App Control.

![sme select](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/sme_select.png)

Figure 8. Open Workspace

Navigate to the directory where you just have started the SME and into the "workspaces":

![workspaces in dir](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/workspaces_in_dir.png)

Figure 9. Workspaces within the Modeling Environment Directory

Now select the workspace folder and the SME will switch from the "Home Screen" to present the selected workspace content within the "Workspace Explorer".

![SME basic](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/SME_basic.png)

Figure 10. SME initial view after opening a Workspace

The different Workspace Items will open a respective Editor on the right, e.g. when selecting a Document Model, the Document Model Editor opens:

![SME DMM](https://geta12.com/docs/2025.06/ext5/overall/quick_start_guide/assets/SME_DMM.png)

Figure 11. SME Document Model Editor

## Next Steps

Now that you’ve seen the tools and Sample Workspaces, here are your next steps:

* Continue reading the *[Preview App](https://geta12.com/docs/overall/preview_app/index.html)* chapter of the Modeling Documentation to find more detailed information about the delivered Sample Workspaces and
* learn more about the *[Simple Model Editor](https://geta12.com/docs/sme/sme-ba-docs/index.html)* to create and edit your own models.
