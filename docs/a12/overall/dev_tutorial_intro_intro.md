# Intro Tutorial

|  |  |
| --- | --- |
|  | This tutorial uses A12 version 2025.06-ext5 and is based on the Project Template version 202506.5.1. |

## Prerequisites

For this introductory tutorial, no prior knowledge of any technologies is necessary.

You only need to have the following programs available:

* Text editor / IDE
* [A12 Modeling Environment](https://geta12.com/installer/) in version **2025.06-ext5**

## Overview

This tutorial serves as an introduction into modeling and how to start a project with the Project Template.

The following aspects will be covered in the tasks:

* [Task 1: Modeling](https://geta12.com/docs/overall/dev_tutorial_intro_modeling/index.html)
* [Task 2: Initializing the Project Template](https://geta12.com/docs/overall/dev_tutorial_intro_project_template/index.html)
