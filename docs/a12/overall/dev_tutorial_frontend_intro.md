# Frontend Tutorial

|  |  |
| --- | --- |
|  | This tutorial uses A12 version 2025.06-ext5 and is based on the Project Template version 202506.5.1. |

## Prerequisites

For this frontend tutorial, prior knowledge of the following technologies is necessary:

* TypeScript
* React
* Redux

  + Redux-Saga

The tutorial also assumes you have the following programs available:

* Text editor / IDE
* [A12 Modeling Environment](https://geta12.com/installer/) in version **2025.06-ext5**
* [redux-devtools](https://github.com/reduxjs/redux-devtools)

We are extending the CRM system, which has been created and implemented in the previous tutorials. If you want to know how it was implemented, you can check them out.

## Overview

This tutorial serves as an introduction into customizing the frontend of an A12 application.

The following aspects will be covered in the tasks:

* [Task 1: Application Frame](https://geta12.com/docs/overall/dev_tutorial_frontend_application_frame/index.html)
* [Task 2: Overview Customization](https://geta12.com/docs/overall/dev_tutorial_frontend_overview_customization/index.html)
* [Task 3: Form Customization](https://geta12.com/docs/overall/dev_tutorial_frontend_form_customization/index.html)
* [Task 4: Data in Activities](https://geta12.com/docs/overall/dev_tutorial_frontend_data_in_activities/index.html)
* [Task 5: Middleware and Saga](https://geta12.com/docs/overall/dev_tutorial_frontend_middleware_and_saga/index.html)
* [Task 6: Unit Testing](https://geta12.com/docs/overall/dev_tutorial_frontend_unit_testing/index.html)
